Front-End Tools

ReactJS Release: 2013 ReactJS is an open-source (with caveats) JavaScript library for building user interfaces.
AngularJS Release: 2010 AngularJS (commonly referred to as AngularJS 1.X) is a JavaScript framework for building SPAs with an MVC architechture.
VueJS Release: 2014 VueJS (pronounced /vjuÀê/, like view) is an open-source progressive JavaScript framework for building user interfaces.
Elm Release: 2016 Elm is a domain-specific functional programming language for declaratively creating web browser-based graphical user interfaces.
Polymer Release: 2015 Polymer is an open-source JavaScript library for building web applications using Web Components.

Back-End Tools

Ruby on Rails Release: 2015 Ruby on Rails, or Rails, is a server-side web application framework written in Ruby. Rails is an MVC framework, providing default structures for a database, a web service, and web pages.
NodeJS Release: 2009 Node.js is a JavaScript runtime built on Chrome's V8 JavaScript engine. Uses an event-driven, non-blocking I/O model that makes it lightweight and efficient.
ASP.NET Core Release: 2016 ASP.NET Core is the next generation of ASP.NET. It is a modular framework that runs on both the full .NET Framework, on Windows, and the cross-platform .NET Core.
Django Release: 2005 Django is a free and open-source web framework, written in Python, which follows the model-view-template (MVT) architectural pattern.
